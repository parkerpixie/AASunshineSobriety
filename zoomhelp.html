Host

Claiming Host

Use this if you need host controls for screen sharing, muting, or keeping the meeting room running smoothly.

Steps to claim host
  • Select Participants at the bottom of your Zoom screen, next to Chat.
  • Select Claim host.
  • Enter the host code when prompted.
  • You will need the host passcode if you want to share your screen or mute participants.

Leader note: Keep the host code private. Do not post it in public chat.

Share

Sharing Your Screen

Use this when you need to share a reading, reflection, or other meeting resource.

Steps to share your screen
  • Open the page you want to share before you try to share your screen.
  • Make sure you have claimed host if sharing is restricted.
  • In Zoom, select the green Share button.
  • Choose the correct page or window.
  • Select Share.
  • The window must be open in the background. It cannot be minimized.
  • When done, select Stop Share in the red box at the top of your screen.

Chat

Copying and Pasting Into Zoom Chat

The main script page has copy buttons for Announcements Part 1 and Part 2.

Copy / paste steps
  • Copy only one announcement section at a time.
  • Paste Announcements Part 1 first.
  • Then paste Announcements Part 2.
  • If Zoom does not send the message, split the section into smaller pieces and try again.

Tip: On Windows, use Ctrl + V. On Mac, use Command + V.

Help

Quick Troubleshooting

If screen sharing does not work
  • Confirm you claimed host.
  • Confirm the window you want to share is not minimized.
  • Try sharing your full screen instead of one window.
If chat paste fails
  • Try pasting a smaller section.
  • Paste Part 1 and Part 2 separately.
  • Check that the cursor is inside the Zoom chat box before pasting.
